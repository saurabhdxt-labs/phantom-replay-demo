# PHANTOM — GPS Jamming Forecast, Proven Daily

A live, self-contained demonstration of the PHANTOM GPS-interference forecaster.

Every point on the map was **forecast 12 hours ahead**, then checked against what
actually happened. Verified hits, false alarms, and misses are all shown — nothing
is cropped to flatter the score. Published with a one-day lag.

**Open the demo:** enable GitHub Pages on this repo (Settings → Pages → Deploy from
branch `main`, folder `/`), then visit `https://<owner>.github.io/<repo>/`.

This page ships only what you see — forecast position, risk, and observed outcome.
No system internals, methodology, or code are included — only the published outcomes.
